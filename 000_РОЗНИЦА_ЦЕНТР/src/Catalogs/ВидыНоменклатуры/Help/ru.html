Справочник "Виды номенклатуры"

Справочник "Виды номенклатуры" предназначен для хранения настроек различных видов номенклатуры, продающейся в магазине.

Нормативно-справочная информация

Настройка ведения информации о номенклатуре

Реквизиты:

Наименование

Наименование вида номенклатуры. (Строка)

Группа

Группа, в которую входит данный вид номенклатуры. (Виды номенклатуры)

Использование характеристик

Вариант использования характеристик для данного вида номенклатуры. Можно выбрать значения: Не использовать, Использовать общие для вида номенклатуры, Использовать индивидуальные для номенклатуры. (Не использовать/Общие для вида номенклатуры/Индивидуальные для номенклатуры)

Тип номенклатуры

Указывается тип, к которому относится номенклатура данного вида. Можно выбрать значения: товар, услуга, подарочный сертификат, комплект (Товар/Услуга/Набор-комплект/Подарочный сертификат)

Шаблон ценника

Шаблон, который будет использоваться по умолчанию при печати ценников. (Хранилище шаблонов)

Шаблон этикетки

Шаблон, который будет использоваться по умолчанию при печати этикеток. (Хранилище шаблонов)

Набор свойств

Набор свойств, которым будет обладать номенклатура с этим видом. (Наборы дополнительных реквизитов и сведений)

Набор свойств характеристик

Набор свойств, которым будет обладать характеристики с этим видом номенклатуры,
если для используются характеристики для вида номенклатуры (Наборы дополнительных реквизитов и сведений)
