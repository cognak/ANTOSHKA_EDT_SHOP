Справочник "Упаковки номенклатуры"

Назначение

Справочник предназначен для хранения перечня упаковок товара. Упаковка может использоваться в качестве дополнительной единицы измерения, и связана с базовой единицей определенным коэффициентом пересчета. Базовая единица определяется для каждой позиции номенклатуры в справочнике Номенклатура (поле Единица измерения).  

Перечень всех используемых на торговом предприятии упаковок задается в справочнике Единицы измерения. Список упаковок, который может быть использован для товара, зависит от того варианта использования упаковок, которое установлено в справочнике Виды номенклатуры.   

Ввод новой упаковки

Порядок ввода новой упаковки зависит от заданного варианта использования упаковок.

  • Использовать общие для вида номенклатуры. В этом случае упаковки вводятся из справочника Виды номенклатуры. Для каждого вида номенклатуры вводится свой список упаковок.
  • Использовать индивидуальные для номенклатуры. В этом случае упаковки вводятся по номенклатурной позиции, зарегистрированной в справочнике Номенклатура.  

Для новой упаковки следует заполнить следующие поля:

  • Наименование. Наименование упаковки номенклатуры.
  • Номенклатура. Вид номенклатуры или позиция номенклатуры, для которой вводится упаковка.
  • Единица измерения. Единица измерения, которой соответствует данная упаковка. Выбирается из справочника Единицы измерения, в котором хранится список всех единиц измерения, которые используются в торговом предприятии.
  • Коэффициент. Количество единиц в одной упаковке. Указывается по отношению к той базовой единице измерения, которая выбрана для позиции номенклатуры.
