Справочник "Наборы упаковок"

Справочник "Наборы упаковок" предназначен для ведения различных упаковок, в которых товар может поступать в магазин. В справочник "Наборы упаковок" можно добавлять любое количество наборов упаковок. Для каждого набора упаковок следует определить свой список упаковок.
