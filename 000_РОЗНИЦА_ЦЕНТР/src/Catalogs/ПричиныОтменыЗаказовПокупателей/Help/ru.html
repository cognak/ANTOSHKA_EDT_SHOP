Причины отмены заказов покупателей

Справочник используется для классификации причин отмены заказов покупателей.

Список открывается по ссылке Администрирование – Продажи - Причины отмены заказов покупателей.С помощью сервиса Найти, который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.
