Справочник "Политики учета серий"

Справочник служит для формирования различных правил ведения учета серий и сроков годности номенклатуры. Использование серий для номенклатуры задается в виде номенклатуры. Правила должны быть сформированы для всех видов номенклатуры, по которым необходимо вести учет серий или сроков годности. Информация о правилах учета серий для вида номенклатуры может быть детализирована для каждого склада. Информация о том, какая политика учета серий применяется на конкретном складе задается в карточке вида номенклатуры.

Указывать по факту отбора

Серии необходимо указать при отгрузке, после того как товары будут отобраны (на неордерном складе - в накладной в статусе "Отгружено", на ордерном - в ордере в статусе "Отгружено") (Булево)

Указывать при отгрузке

Требовать указание серий при отгрузке со склада (Булево)

Указывать при отгрузке в розницу

Требовать указание серий при отгрузке товаров клиентам (Булево)

Указывать при отгрузке клиенту

Требовать указание серий при отгрузке товаров клиентам (Булево)

Указывать при отгрузке комплектов для разборки

Требовать указание серий при отгрузке со склада комплектов для последующей разборки их на комплектующие (Булево)

Указывать при отгрузке комплектующих для сборки

Требовать указание серий при отгрузке со склада комплектующих для сборки (Булево)

Указывать при отгрузке на внутренние нужды

Требовать указание серий при отгрузке со склада товаров на внутренние нужды компании или по распоряжению на списание (Булево)

Указывать при отгрузке по возврату поставщику

Требовать указание серий при отгрузке со склада по возврату товаров поставщику (Булево)

Указывать при отгрузке по перемещению

Требовать указание серий при отгрузке со склада по перемещению на другую складскую территорию (Булево)

Указывать при отражении излишков, недостач и порчи

Требовать указание серий при пересчете товаров, отражении излишков, недостач и порчи (Булево)

Указывать при перемещении между помещениями

Требовать указаывать серии при перемещении товаров между помещениями одной складской территории (Булево)

Указывать при планировании отбора

Серии необходимо указать до начала отгрузки (на неордерном складе в накладной в статусе "К отгрузке", на ордерном - в расходном ордере в статусе "К указанию серий") (Булево)

Указывать при планировании отгрузки

Серии необходимо указать до начала отгрузки (т.е. в накладной в статусе "К отгрузке") (Булево)

Указывать при приемке по перемещению

Учет серий по FEFO

Серии подбираются системой автоматически по срокам годности при подготовке отбора товаров (Булево)

Учитывать остатки серий

Программа будет вести складские остатки по сериям, контролируя правильность указания серий при отгрузке (Булево)
