Справочник Работа выполняемая сотрудниками.

В справочнике хранится информация о работах, которые используются при планировании рабочего времени сотрудников магазинов.

Список элементов открывается по ссылке Персонал – Работа магазинов - Работы выполняемые сотрудниками. С помощью сервиса Найти, который запускается из командной панели списка, Пользователь может выбрать нужные строки, задавая поиск по значениям различных реквизитов.

Новую запись в справочник можно добавить с помощью кнопки Создать в командной панели списка.
