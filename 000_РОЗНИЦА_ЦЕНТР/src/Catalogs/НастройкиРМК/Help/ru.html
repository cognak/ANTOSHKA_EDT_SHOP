Справочник "Настройки РМК"

В этом справочнике хранятся настройки рабочего места кассира (РМК).

Предусмотрены следующие настройки РМК:

  • Объединять позиции с одинаковым товаром. Этот флаг следует устанавливать в том случае, если не применяется построчная печать товаров чека на фискальном регистраторе.
  • Назначать цену товарам с нулевой ценой. Если этот флаг установлен, то это означает, что если на товар не заданы цены магазина, то они могут быть отредактированы вручную кассиром в чеке.
  • При возврате распечатывать пакет документов.  Этот флаг следует устанавливать в том случае, если при регистрации факта возврата товаров от покупателя, кассир должен иметь возможность распечатать необходимые документы, сопровождающие возврат товаров: заявление покупателя (или кассира) о причине возврата, акт возврата, расходный кассовый ордер.
  • Установка флага Контролировать остатки товаров при закрытии чека позволяет контролировать остатки товара в момент проведения документа Чек ККМ.
  • Установка флага Выбор информационной карты только по коду закрывает доступ кассира к общему списку информационных карт сотрудников. Идентификация сотрудника производится при вводе кода его информационной карты без показа кассиру общего списка информационных карт всех сотрудников.
  • Установка флага Проводить опрос владельца карты активизирует функционал регистрации данных покупателя и дисконтной карты при оформлении покупки. Для введения данных в режиме РМК, нужно настроить дополнительные права кассира.
  • Флаг Округлять процент скидки в большую сторону применяется при ручном назначении суммовой скидки. В программе предусмотрена возможность назначения суммовой скидки по каждой позиции номенклатуры при оформлении чека. Суммовая скидка может назначаться по цене или по общей сумме товара. При этом процент скидки вычисляется как сумма скидки, деленная на цену или сумму товара. Процент скидки округляется в большую или меньшую сторону, в зависимости от установленной настройки.
  • На закладке Интерфейс определяется тот интерфейс, который представляется для работы кассиру.
    • Если установлен флажок Отображать текстовый подбор в правой части экрана, тогда пользователю предоставляется интерфейс с использованием стандартного диалогового окна подбора товаров. Работа в РМК осуществляется с использованием клавиатуры и мыши, с возможностью подбора товара по текстовому представлению. Обычно такой режим используется при торговле непродовольственными товарами (непродовольственная розница).
    • Если установлен флажок Отображать клавиатуру в правой части экрана, то для работы используется РМК с программируемой клавиатурой (без использования мыши). В правой части рабочего места кассира отображаются кнопки для удобного нажатия с touch screen или клавиатуры. Интерфейс адаптирован под потоковое сканирование товаров. Обычно такой режим работы применяется в магазинах, торгующих продовольствием (продовольственная розница). Для данного режима работы РМК настраиваются сервисные функции по подбору товаров (закрытие подбора при выборе товара, определение поля поиска по умолчанию (код, артикул или штрихкод товара)).
  • На закладке Быстрые товары определяется список быстрых товаров. Для товаров, которые продаются достаточно часто, можно назначить кнопку быстрого выбора (кнопка товара) и сочетание клавиш (акселератор), при нажатии которых товар будет быстро выбран из списка. 
  • На закладке Права пользователей устанавливаются права кассира по оформлению продаж. Данная возможность доступна только пользователю, который имеет права администратора. Администратор может разрешить или запретить кассиру выполнять операции по аннулированию чека, оформлению чека на возврат, снятие z-отчета и т.д. То есть с помощью возможностей настройки прав можно разделить обязанности между кассирами и администраторами касс (старшими кассирами).
  • При установленом флаге Использовать расширенную настройку нижней панели на закладке  Общие настройки становится доступна закладка Нижняя панель, на которой  настраиваются состав и порядок отображения кнопок управления функциями РМК. В табличной части Пользователь может задать Максимальное количество кнопок в одном ряду. Кнопки заполняют панель слева направо, сверху вниз. Пользователь может заполнить список команд по умолчанию или подобрать команды вручную из выпадающего списка. Порядок кнопок нижней панели управляется стрелочками вверх или вниз на командной панели закладки.  Пользователь может назначить команде акселератор. Поле "Наименование кнопки" формируется автоматически как стандартное наименование плюс акселератор.
  • Кнопка Настройка торг.оборудования позволяет непосредственно из формы настройки РМК перейти в форму подключение и настройка торгового оборудования, чтобы подключить к рабочему месту кассира необходимое торговое оборудование: контрольно-кассовую машину , сканер штрих-кода, считыватель магнитных карт, электронные весы и т.д.
