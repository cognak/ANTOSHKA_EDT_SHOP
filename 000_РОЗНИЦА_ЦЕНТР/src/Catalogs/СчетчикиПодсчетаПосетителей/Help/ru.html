Справочник "Счетчики подсчета посетителей"

    Справочник предназначен для регистрации счетчиков прохода посетителей. Для каждого счетчика пользователь вводит реквизиты элемента справочника.
     
    Наименование, позволяющее идентифицировать счетчик.
    Код
    Магазин – ссылка на элемент справочника Магазины, обязательный реквизит.
    Технический код – это строковое значение для определения счетчика при загрузке из файла. Если загрузка осуществляется из файла с форматом данных производителя, то это код счетчика в формате производителя. Например, «{022862E9-9BBE-4875-9E08-7C9DA59DD3E2}». Если загрузка осуществляется из файла определенного формата, то это любой строковый код, который будет использоваться в файле данных.
    Вид счетчика – реквизит, определяющий логику работы счетчика:
  • двунаправленный счетчик, умеющий определять направление движение посетителя
  • однонаправленный – Вход счетчик, установленный на входе в торговый зал и регистрирующий только пересечение линии прохода
  • однонаправленный – Выход счетчик, установленный на выходе из торгового зала и регистрирующий только пересечение линии прохода
  • однонаправленный – Вход-выход счетчик, регистрирующий только факт пересечения линии прохода без учета направления движения.
     
    В табличной части Статистика подсчета сотрудников пользователь может внести данные о среднем количестве проходов сотрудников торгового зала, если это нужно учитывать в статистике. Например, магазин в торговом центре может иметь единственный вход в торговый зал. Можно учесть сдвиги статистики определив время регистрации прихода-ухода сотрудников на работу, на обед и количество проходов, которое нужно вычесть из данных счетчика.
