Регистрация выдачи денежных средств.

С помощью Расходного кассового ордера регистрируется выдача денежных средств на размен в кассу ККМ, выдача заработной платы сотрудникам, оплату хозяйственных нужд, оплаты поставщикам и возвраты покупателям после закрытия кассовой смены. Выплаты регистриру.ются в разрезе предустановленных хозяйственных операций, аналитика которых может быть расширена с помощью Статей учета движений денежных средств.

Как оформить оплату за поставку?

Чтобы оформить оплату поставщику нужно в списке документов поступления выбрать нужный документ и нажав кнопку Создать на основании выбрать пункт Расходный кассовый ордер. При этом реквизиты Организация, Контрагент, Касса и Сумма к оплате заполняется из документа–основания. Пользователь должен выбрать одну из статей ДДС, относящихся к операции оплата поставщику. Если оплате подлежат несколько документов, то их можно добавить в табличную часть. Сумма к оплате при этом пересчитается.

Как оформить возврат наличной оплаты покупателю?

Выдача денежных средств покупателю при возврате покупки после окончания кассовой смены производится в операционной кассе магазина. Для оформления возврата выберите нужный документ в списке Возвраты от покупателя и в меню Создать на основании выберите пункт Расходный кассовый ордер. При этом Организация, Контрагент, Сумма документа заполнятся из документа-основания, а Кассу пользователь может выбрать из числа разрешенных ему касс организации.

Документ можно также Создать из списка Расходных кассовых ордеров, выбрав операцию Возврат ДС от покупателя. В этом случае реквизиты заполняются вручную, а документ-основание указывать не обязательно. В качестве покупателя может быть указан как контрагент, так и розничный покупатель.

По оформленному и проведенному документу может быть напечатан чек на подключенном к данному рабочему месту оборудовании - фискальном принтере или принтере документов. Для этого нужно нажать кнопку Напечатать чек в командном меню документа.

Как оформить инкассацию в банк?

Для оформления сдачи денежных средств в банк нажмите кнопку Создать в списке Расходные кассовые ордера (раздел Финансы) и выберите операцию Сдача ДС в банк.

В открывшейся форме пользователь вводит Кассу из числа разрешенных касс. Организация определяется автоматически. Пользователь выбирает Счет организации в банке, накоторый будут зачислены инкассированные средства, и заполняет поле Номер сумки инкассатора. В табличной части закладки Опись сдаваемых наличных денег кассир указывает количество банкнот каждого номинала. Сумма инкассации определяется по описи.

На закладке Печать можно заполнить поля, необходимые для печати Расходного кассового ордера. Помимо ордера можно также распечатать Опись ценностей в нац. валюте (Приложение 4) и Препроводительную ведомость к сумке (Приложение 6).

Как оформить передачу ДС в другую кассу предприятия?

Передача денежных средств между кассами предприятия производится в два этапа:

  • в кассе-отправителе оформляется Расходный кассовый ордер на нужную сумму;
  • после фактического поступления денег в кассе получателе оформляется Приходный кассовый ордер на основании Расходного документа.

Передача денежных средств между кассами одной организации производится по операции Выдача ДС в другую кассу. При этом Организация документа определяется по кассе-отправителю, а в качестве кассы-получателя можно выбрать Кассы только этой организации.

Если денежные средства должны быть переданы между организациями, то нужно создать РКО с операцией Выдача денежных средств в другую организацию. Для данной операции никаких ограничений на выбор кассы-получателя не накладывается.

Как оформить выдачу разменной монеты в кассу ККМ?

Выдача разменной монеты оформляется с помощью операции Выдача ДС в кассу ККМ. При этом организация определяется по Кассе-отправителю и в качестве получателя размена можно указать только Кассы ККМ определенной организации. По проведенному документу РКО на рабочем месте кассира (РМК) производится операция Внесение.

Как оформить другие расходы из кассы магазина?

Для оформления расходов денежных средств по другим нуждам предусмотрена операция Прочие расходы. При этом предприятие имеет возможность анализировать прочие расходы с нужной для руководства предприятием детализацией, используя справочник Статьи движения денежных средств.

Регистрация поступлений наличных денежных средств.

С помощью Приходного кассового ордера можно зарегистрировать поступление розничной выручки из кассы ККМ, денежных средств в оплату за продажу мелким оптом, возврат денежных средств от поставщика, поступления разменной монеты из банка. Поступления регистрируются в разрезе предустановленных хозяйственных операций, аналитика которых может быть расширена с помощью Статей учета движений денежных средств.

Как найти Расходный кассовый ордер?

Список приходных кассовых ордеров открывается из меню Денежные средства раздела Финансы.

Для удобства работы Пользователь может установить отбор по полям Организация, Касса. По каждому документу в списке дополнительно выводится информация о Дате и Опреации и Сумме документа.

На закладке Денежные средства к поступлению можно просмотреть документы передачи денежных средств между кассами предприятия, по которым ожидается поступление.

С помощью команды Печать для каждого выбранного документа может быть распечатат Приходный кассовый ордер.

На основании Приходного ордера может быть создан только Расходный кассовый ордер.

Как оформить наличную оплату за реализованный мелким оптом товар?

Чтобы оформить оплату по реализации нужно в списке документов Реализация товаров (раздел Продажи)выбрать нужный документ и нажав кнопку Создать на основании выбрать пункт Приходный кассовый ордер. При этом реквизиты Организация, Контрагент и Сумма к оплате заполняется из документа–основания. Пользователь должен выбрать одну из разрешенных ему Касс организации, накоторую будутоприходованы средства и Статью ДДС, относящихся к операции Поступление оплаты от клиента. Если оплате подлежат несколько документов, то их можно добавить в табличную часть. Сумма к оплате при этом пересчитается автоматически.

По оформленному и проведенному документу может быть напечатан чек на подключенном к данному рабочему месту оборудовании - фискальном принтере или принтере документов. Для этого нужно нажать кнопку Напечатать чек в командном меню документа.

Как зарегистрировать возврат денежных средств от поставщика при оформлении возвратов товаров?

При создании РКО необходимо выбрать операцию Возврат ДС от поставщика и в открывшейся форме заполнить реквизиты:

  • Касса, - пользователь может выбрать операционную кассу из числа доступных ему касс;
  • Организация заполняется по реквизиту "Организация" операционной кассы;
  • Контрагент - нужный поставщик выбирается из списка контрагентов;
  • Сумма.

Кассир может добавить один или несколько документов Возврат поставщика в табличную часть расходного кассового ордера. При этом сумма РКО пересчитается на основании сумм указанных документов Возврата поставщику. Для каждого документа Кассир должен выбрать из справочника статью учета ДДС.

Кассир может также по необходимости заполнить текстовые поля Основание, Приложение , Комментарий.

Если приходный кассовый ордер оформляется по одному документу возрата, то он может быть оформлен непосредственно из документа "Возврат поставщику".

Как оформить поступление из банка?

Для оформления поступления денежных средств из банка, например, разменной монеты, нажмите кнопку Создать в списке Приходные кассовые ордера (раздел Финансы) и выберите операцию Поступление ДС из банка.

В открывшейся форме пользователь вводит кассу из числа разрешенных Касс. Организация определяется автоматически. Пользователь выбирает Счет организации в банке и Сумму поступления.

Заполнив необходимые для печати поля и воспользовавшись сервисом Печать пользователь может распечатать Приходный кассовый ордер.

Как оформить поступление переплаты от поставщика?

Переплата от поставщика может возникать при неверном расчете с поставщиком или при возврате товара.

В первом случае документ можно создать из списка Приходные кассовые ордера, выбрав операцию Возврат ДС от поставщика. В этом случае реквизиты заполняются вручную, а документ-основание указывать не обязательно.

Поступление денежных средств может быть оформлено на основании документа Возврат товаров поставщику. Для оформления поступления выберите нужный документ в списке Возвраты товаров поставщику (раздел Запасы и закупки) и в меню Создать на основании выберите пункт Приходный кассовый ордер. При этом Организация, Контрагент, Сумма документа заполнятся из документа-основания, а кассу пользователь может выбрать из числа разрешенных ему Касс организации.

Заполнив необходимые для печати поля и воспользовавшись сервисом Печать пользователь может распечатать Приходный кассовый ордер.

Как оформить поступление из другой кассы предприятия?

Поступления денежных средств из другой кассы предприятия можно оформить на основании Расходного кассового ордера. В форме списка Приходные кассовые ордера раздела Финансы перейдите на закладку Денежные средства к поступлению, выберите нужный документ и нажмите кнопку Принять оплату в командном меню списка.

На основании Расходного кассового ордера сформируется Приходный кассовый ордер с заполненными реквизитами:

  • Для РКО с операцией Выдача ДС в другую кассу – ПКО с операцией Поступление ДС из другой кассы;
  • Для РКО Выдача ДС в другую организацию – ПКО с операцией Поступление ДС из другой организации;
  • Для документа Выемка денежных средств из кассы ККМ – ПКО с операцией Поступление ДС из кассы ККМ.

Пользователь может отредактировать сумму поступления, если фактическая сумма поступления не соответствует расходному документу.

Учет оплат по платежной карте.

С помощью платежной кассы могут быть оплачены как розничные покупки, так и отгрузки мелким оптом.

Оплата розничных покупок производится в рабочем месте кассира в специализированном интерфейсе. Оплата отгрузки мелким оптом производится с помощью документа Эквайринговая операция, которая может быть двух видов: Поступление оплаты от покупателя и Возврат оплаты покупателю.

Список документов доступен в меню Эквайринг раздела Финансы при установленной настройке параметров учета Использовать оплаты платежными картами.

Как найти оплату мелкооптового покупателя, произведенную пластиковой картой?

Для удобства работы Пользователь может установить отбор по полям Магазин, Организация, Терминал. По каждому документу в списке выводится информация о Дате и Сумме документа, Договоре эквайринга и начисленной по оплате Комиссии.

Печать квитанции о проведенной операции проводится на эквайринговом терминале. Если оборудование подключено к рабочему месту, то данные о сумме опалты передаются автоматически при нажатии кнопки Оплатить картой. В противном случае пользователь вводит сумму оплаты вручную, используя клавиатуру эквайрингового терминала.

Для создания документа воспользуйтесь кнопкой Создать командного меню списка и выберите операцию: Поступление оплаты от покупателя или Возврат оплаты покупателю.

Как зарегистрировать оплату с помощью платежной карты?

Во вновь созданном документе нужно заполнить Магазин, выбрать Эквайринговый терминал из числа установленных в магазине. Договор эквайринга и Банк-эквайрер определятся автоматически. Пользователь выбирает один из доступных по договору Видов оплаты и вводит Номер карты выбранного из справочника Контрагента.

Сумма оплаты вводится в поле табличной части документа с указанием Статьи движения денежных средств. Комиссия банка рассчитывается автоматически согласно условиям договора.

В табличной части можно также указать один или несколько документов, которые необходимо оплатить. Сумма к оплате рассчитаются автоматически по суммам документов.

При проведении документа уменьшается задолженность покупателя на зафиксированную сумму оплаты и на эту же сумму увеличивается задолженность банка-эквайрера. Все взаиморасчеты с банком-эквайрером осуществляются в управляющей системе.

Как оплатить платежной картой мелкооптовую продажу?

Документы оплаты платежной картой могут быть также созданы на основании документов "Реализация товаров". В этом случае данные о магазине, организации и контрагенте, а также сумма к оплате заполняются из документов-оснований. Пользователь корректирует по необходимости статью ДДС, вводит вид оплаты и номер карты покупателя и производит оплату, нажав на кнопку "Оплатить картой".

Как оформить возврат на платежную карту клиента?

Документы возврата платежа на платежную карту создаются на основании документов "Возврат товара от покупателя". В этом случае данные о магазине, организации и контрагенте, а также сумма к оплате заполняются из документа-основания. Пользователь корректирует по необходимости статью ДДС, вводит вид оплаты и номер карты покупателя и производит оплату, нажав на кнопку "Оплатить картой".

Возврат на платежную карту клиента можно так же, как и документ "Оплата по платежной карте" создать вручную, только нужно выбрать операцию Возврат оплаты от покупателя.

Как проследить движения сертификатов по сети магазинов?

Всю информацию о передвижениях сертификатов по сети магазинов можно получить с помощью отчета Ведомость движения подарочных сертификатов. Отчет предоставляет информацию за указанный пользователем период в разрезе складов, подарочных сертификатов, серийных номеров. Пользователь может вывести в отчет дополнительные поля:

  • Номинал,
  • Дата реализации,
  • Дата погашения,
  • Дата списания,
  • Дата окончания действия.

Показатели отчета:

  • Начальный остаток (количество) – остаток сертификатов на складе на начало отчетного периода;
  • Сумма начального остатка по номиналу - (Начальный остаток (количество) * Номинал сертификата);
  • Приход (количество) – количество поступивших на склад сертификатов;
  • Сумма прихода по номиналу - (Приход (количество) * Номинал сертификата);
  • Расход (количество) – количество списанных со склада сертификатов;
  • Сумма расхода по номиналу - (Расход (количество) * Номинал сертификата);
  • Конечный остаток (количество) – остаток сертификатов на складе на конец отчетного периода;
  • Сумма конечного остатка по номиналу - (Конечный остаток (количество) * Номинал сертификата).

Как просмотреть движения Подарочных сертификатов по серийным номерам?

Историю движений сертификата можно просмотреть с помощью Отчета о движениях сертификатов. Отчет строится в разрезе подарочного сертификата, серийного номера. По каждому документу движения сертификата выводится информация:

  • дата операции,
  • номинал сертификата,
  • хозяйственная операция,
  • отправитель,
  • получатель.

Как составить отчет по подарочным сертификатам для эмитента?

В Отчете эмитенту по Подарочным сертификатам предоставляется информация о реализации и погашении подарочных сертификатов.

Пользователю предоставляется информация о проданных сертификатах в разрезе организации, магазина, подарочного сертификата, серийного номера. Дополнительно пользователь может установить группировку по складу магазина и вывести номинал сертификата. Показатели отчета:

  • Количество – количество реализованных сертификатов.
  • Сумма - Номинал * Количество.

В качестве расшифровки по каждому сертификату можно получить отчет в разрезе серийных номеров с дополнительными полями:

  • Дата начала действия подарочного сертификата,
  • Дата окончания действия подарочного сертификата.

Как выявить случаи превышения номинала сертификата над суммой чека?

В ситуации, когда покупаель оплачивает сертификатом покупку с суммой меньше номинальной стоимости сертификатов образуется дополнительная прибыль. Проанализировать эту ситуацию можно с помощью отчета Анализ погашения сертификатов.

Отчет строится в разрезе организации, магазина, чека ККМ. В качестве дополнительного поля выводится дата чека ККМ. Показатели отчета:

  • Сумма чека – сумма чека, оплаченного подарочным сертификатом.
  • Сумма подарочного сертификата – если сертификатов несколько, то сумма сертификатов.
  • Внереализационная прибыль - (Сумма подарочного сертификата - Сумма чека).
