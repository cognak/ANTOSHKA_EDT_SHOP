Документ "Списание товаров"

С помощью документа "Списание товаров" оформляется списание не подлежащего продаже и хранению товара и фиксируется недостача на основании проведенной инвентаризации.





Если документ формируется для списания некондиционного товара, можно указать причину списания, выбрав из списка сформированных заранее хозяйственных операций. Анализ причин списания номенклатуры можно произвести с помощью отчета "Причины списания".

Списание товаров может производиться по выбранному пользователем типу цен.

В том случае, если документ оформляется на основании документа Инвентаризация товаров, его табличная часть автоматически заполняется в соответствии с данными, указанными в инвентаризации. Цены в этом случае устанавливаются в соответствии с ценами, указанными в документе инвентаризации.

Печать документа
Для печати документа необходимо нажать на кнопку Печать и выбрать ту печатную форму, по которой печатается документ.

Одну из печатных форм можно установить, как форму по умолчанию (кнопка По умолчанию).
Если документ печатается без предварительного просмотра, то нужно установить флаг На принтер.
Если необходимо внести изменения в печатную форму документа:

Документ выводится для предварительного просмотра (флаг На принтер не установлен).
В сформированной печатной форме необходимо нажать на кнопку Только просмотр и внести изменения в печатную форму документа.
Нажать на кнопку Печать и распечатать документ.


Сервисные возможности
При оформлении документов предусмотрены сервисные возможности (кнопка Сервис на панели инструментов) по заполнению и изменению списка товаров в документе.

Поиск по штрихкоду - в появившемся диалоговом окне можно задать штрихкод товара. Программа найдет товар по штрихкоду и добавит товар в табличную часть документа. Предусмотрено также заполнение табличной части документа с помощью сканера штрихкода.
Заполнить табличную часть - с помощью данной обработки можно заполнить товары одной группы, только товары, имеющиеся на складе, установить цены по новому типу цен и т.д.
Заполнить из терминала сбора данных - табличная часть документа заполняется из терминала сбора данных. Предварительно терминал сбора данных должен быть подключен и настроен.
Изменить табличную часть - с помощью этой обработки можно произвести различные действия с позициями, указанными в табличной части документа: установить ручную скидку на товары, распределить дополнительную сумму на все товары и т.д.
Напечатать ценники - будут распечатаны ценники на список товаров в документе по указанным в документе ценам.
