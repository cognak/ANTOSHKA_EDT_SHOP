Документ "Установка себестоимости"

Документ Установка себестоимости предназначен для ручной корректировке информации о себестоимости товаров.

Данную возможность можно использовать в том случае, если себестоимость товаров можно определить только экспертным путем, например, при переходе на конфигурацию Розница неавтоматизированных магазинов для ввода начальной информации о себестоимости товаров.

В документ вводится информация о товарах и себестоимости единицы товара (Цена). Документ фиксирует одинаковую себестоимость товаров для всех магазинов, зарегистрированных в информационной базе конфигурации Розница.

При оформлении документа предусмотрены сервисные возможности (кнопка Сервис на панели инструментов)  по заполнению и изменению списка товаров в документе.

  • Поиск по штрихкоду - в появившемся диалоговом окне можно задать штрихкод товара. Программа найдет товар по штрихкоду и добавит товар в табличную часть документа. Предусмотрено также заполнение табличной части документа с помощью сканера штрихкода.
