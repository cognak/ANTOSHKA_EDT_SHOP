Документ "Сводный отчет по кассовой смене"

Документ предназначен для формирования сводного отчета по магазину. Документ формируется на основе документов Отчет о розничных продажах, ранее сформированных при закрытии кассовой смены по каждой кассе ККМ магазина.

Табличная часть документа заполняется списком документов Отчет о розничных продажах. Табличная часть документа может быть заполнена вручную путем подбора документов Отчет о розничных продажах или с помощью кнопки Создать в командной панели списка документов.

Документ может быть распечатан.
