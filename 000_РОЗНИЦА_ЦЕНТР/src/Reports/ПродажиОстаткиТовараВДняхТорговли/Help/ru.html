Отчет "Продажи, остатки товаров в днях торговли"

Отчет "Продажи, остатки товаров в днях торговли" объединяет в себе информацию о продажах товаров за указанный период и о текущих остатках товаров на складах. С помощью этого отчета можно оценить то количество дней торговли, на которое хватит остатка товаров, исходя из анализа продаж за указанный в отчете период.  Информация о продажах товаров оценивается как по ценам продажи, так и по их себестоимости.  С помощью отчета можно определить, какую долю занимает в общем объеме продаж продажа товаров определенной группы товаров или конкретной товарной позиции. Также с помощью отчета можно определить, какого количества товара недостаточно для сохранения положительной динамики продаж, а по какому товару имеется избыточный остаток относительно динамики его продажи. Данный отчет также используется в качестве расшифровки отчета "Показатели эффективности".

Настройки отчета

  • Период отчета. В отчете устанавливается период, за который будет предоставлена информация о продажах товаров. Информация об остатках товаров формируется на дату окончания отчета. Для корректного расчета показателей отчета необходимо, чтобы период, который задается в отчете, был завершенным. Завершенным периодом считается тот интервал, у которого дата его окончания меньше либо равна текущей даты. Незавершенный период может быть установлен в том случае, если отчет формируется в качестве расшифровки отчета "Показатели эффективности".
  • Рассчитывать дни торговли относительно показателя продаж. Можно выбрать один из показателей продаж: количество (количество товаров, проданных за период отчета), себестоимость (сумма продажи товаров по его себестоимости) или сумма продаж (сумма продажи товаров в розничных ценах).

В дополнительном диалоговом окне (кнопка Настройка) можно настроить аналитические разрезы (группировки) отчета и произвести отбор данных, которые должны быть представлены в отчете.

  • Аналитические разрезы (группировки) отчета. В отчете можно установить группировку данных по магазину, по номенклатурной группе, по группе (иерархии) номенклатуры, по  конкретной позиции номенклатуры или по списку номенклатурных позиций, а также по характеристикам номенклатуры.
  • Отбор данных. Пользователь  может   устанавливать отборы и упорядочивать данные по любому аналитическому разрезу, а также по любому из показателей, представленных в отчете. 

Показатели отчета


Представленные  в отчете показатели разделены на две группы. В группе  "Продажи" показываются показатели, которые относятся к продажам товаров за указанный период. В группе "Остатки" показываются показатели, которые относятся к остаткам товаров, рассчитанным на дату конца периода отчета.

Продажи

  • Количество (продажи). Количество проданного товара за выбранный период.
  • Ед.изм. (продажи). Единица  измерения товара. В качестве единицы измерения используется единица хранения остатков.
  • Сумма, грн. (продажи).  Общая сумма продаж товаров, зарегистрированная в документах продажи, за выбранный период.  Сумма выводится с учетом всех предоставленных скидок (наценок) при продаже товаров.
  • Цена розничная, грн. (продажи). Средняя цена продажи товара относительно общей суммы продаж.
    • Рассчитывается по формуле: Сумма грн, (продажи)/ Количество (продажи).
  • Сумма себестоимости, грн. (продажи). Сумма себестоимости товаров, проданных за период отчета.
  • Цена себестоимости, грн. (продажи). Средняя себестоимость проданных товаров.
    • Рассчитывается по формуле: Сумма себестоимости, грн. (продажи)/ Количество (продажи)
  • Доход, грн. Доход полученный, при продаже товаров.
    • Рассчитывается по формуле: Сумма грн, (продажи) - Сумма себестоимости, грн. (продажи).
  • Дней торговли. Товарный запас в днях продажи. Показатель показывает на какое количество дней хватит запаса товаров.
    • Расчет данного показателя производится по следующей формуле: 
      Суммарный остаток показателя на конец периода / Средняя сумма продаж показателя за день
      • Средняя сумма продаж показателя за день рассчитывается по следующей формуле:
        Показатель продаж за интервал / Количество дней в интервале.  
        • Показатель продаж за интервал – это тот показатель который выбран в настройках отчета ("Рассчитывать дни торговли относительно показателя продаж"): количество, себестоимость или сумма продаж.
        • Количество дней интервала рассчитывается следующим образом:
          • Если текущий интервал завершенный, тогда рассчитывается количество дней в интервале.
          • Если текущий интервал незавершенный, тогда рассчитывается количество дней в интервале между началом интервала и текущей датой.

Остатки

  • Количество (остатки). Количество остатка товара на дату конца периода, указанного в отчете.
  • Ед.изм. (остатки). Единица  измерения товара. В качестве единицы измерения используется единица хранения остатков.
  • Цена розничная, грн. (остатки). Розничная цена, которая зарегистрирована в магазине, для данной товарной позиции на дату, равную дате конца периода, указанного в отчете.
  • Сумма розничная, грн. (остатки). Сумма товара по розничной цене, установленной в магазине на конец периода.
    • Рассчитывается по формуле: Количество (остаток)* Цена розничная, грн (остаток).
  • Цена себестоимость, грн. (остатки). Цена, которая зарегистрирована в документе последнего поступления товара, оформленного на дату, равную концу периода, установленного в отчете.
  • Сумма себестоимость, грн. (остатки). Сумма товара по цене себестоимости, установленной для товара на конец периода.
    • Рассчитывается по формуле Количество (остаток)* Цена себестоимость, грн (остаток). 

Важно. Расчет себестоимости зависит от установленного в учетной политике способа учета себестоимости. Показатели, относящиеся к себестоимости товаров, выводятся только в том случае, если в программе установлен один из способов учета себестоимости : Автономный учет или Принимать из управляющей системы.  Если способ учета установлен "Не учитывать", то значения показателей себестоимости в отчете устанавливаются "Не определено".
