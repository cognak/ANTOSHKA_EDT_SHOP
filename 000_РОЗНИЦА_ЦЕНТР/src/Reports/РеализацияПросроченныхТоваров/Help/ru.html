Отчет "Реализация просроченных товаров"

Отчет позволяет просмотреть серии товаров, реализованные после истечения срока годности. Для формирования отчета необходимо, чтобы при оформлении продаж товара вводились серии.
При формировании отчета для каждого товара выводятся серии датами продаж, превышающими срок годности, установленный для серии.
