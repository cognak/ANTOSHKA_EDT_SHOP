Отчет "Показатели эффективности"

Отчет Показатели эффективности предназначен для мониторинга и анализа основных показателей деятельности магазина.

При настройке данного отчета не применяются стандартные настройки отчетов. Группировка данных в отчете может производиться по одному из трех возможных вариантов (Разрез аналитики): Магазин, Номенклатурная группа или Иерархия номенклатуры.

Программа предоставляет возможность пользователю выбрать дату и интервал статистики: неделя, декада, месяц. На основе указанной пользователем даты и указанного интервала статистики автоматически рассчитываются даты текущего интервала и предыдущего интервала, за который будет производиться анализ продаж. Текущим считается интервал, включающий в себя указанную пользователем дату.

При расчете динамики сравниваются данные текущего и предыдущего интервалов, в связи с этим, для корректного расчета динамики интервалы должны быть завершенные. Завершенным интервалом считается тот интервал, у которого дата его окончания меньше либо равна текущей даты. Незавершенным интервалом считается тот интервал, у которого дата окончания больше текущей даты, при этом сам текст интервала на форме отчета выделяется красным цветом.

Пример. Пользователь указал дату 12.06.2008 года и интервал месяц. Тогда в качестве текущего интервала будет задан июнь, а в качестве предыдущего интервала для расчета данных по изменению (приросту или падению различных показателей) данных будет использован месяц май.

Для сравнения могут быть использованы также показатели аналогичного периода предыдущего года (флажок Аналогичный период за предыдущий год).

Информация о некоторых показателях может быть выведена в виде диаграммы. Для показа данных в диаграмме необходимо задать количество интервалов, за которые выводятся данные предыдущих периодов. Текущий месяц входит в количество интервалов.

Пример. Пользователь указал дату 12.06.2008 года, интервал – месяц, количество интервалов – 4. Тогда в диаграмме будут выведены данные с детализацией по следующим месяцам: март, апрель, май и июнь. 

Отчет разделен на три раздела: Продажи, Запасы и Списание.

По каждому разделу выводятся показатели в заданной группировке: по магазинам, номенклатурным группам, иерархии номенклатуры.

В разделе Продажи выводятся следующие показатели:

  • Количество чеков. Рассчитывается количество заархивированных чеков. При этом не учитываются чеки по возврату товаров. 
  • Средний чек. Сумма среднего чека. Рассчитывается как общая сумма заархивированных чеков, деленная на количество чеков. При этом не учитываются чеки по возврату товаров. 
  • Продажа. Общая сумма всех продаж товаров (с учетом предоставленных скидок).  
  • Доход. Рассчитывается как разница между суммой продажи и себестоимостью на момент продажи. 

В разделе Запасы отображаются следующие показатели:

  • Дней – товарный запас в днях продажи. То есть на какое количество дней хватит запаса товаров.
    • Расчет данного показателя производится по следующей формуле:Суммарный остаток товаров по себестоимости (на конец текущего периода) / Средняя сумма продаж по себестоимости за день.
      • Средняя сумма продаж по себестоимости за день рассчитывается по следующей формуле:
        Сумма продаж по себестоимости за текущий интервал времени / Количество дней текущего интервала.  
      • Количество дней текущего интервала рассчитывается следующим образом :
        • Если текущий интервал завершенный, тогда рассчитывается количество дней в интервале, указанном в отчете.
        • Если текущий интервал незавершенный, тогда рассчитывается количество дней в интервале между началом текущего интервала и текущей или рабочей датой.
  • Продажа (с/с) – суммарная себестоимость продаж товаров за текущий интервал времени.
  • Остаток (с/с) – общая себестоимость всех остатков товаров на конец текущего интервала. 

В разделе Списание отображаются следующие показатели:

  • Итоги инв. – рассчитывается как разница между суммой оприходованных товаров и суммой списанных товаров. Суммы списания и оприходования рассчитываются по документам Оприходование товаров и Списание товаров, оформленных за текущий период. При этом суммы списания и оприходования рассчитываются по тем розничным ценам, которые действовали для товаров в момент списания или оприходования товаров.
  • Доля в продаже – доля итогов инвентаризации в сумме продаж товаров.
    • Рассчитывается по формуле: Итоги. инв. / Продажа * 100 %  
  • Доля в доходе – доля итогов инвентаризации в доходе торгового предприятия.
    • Рассчитывается по формуле: Итоги. инв. / Доход * 100 %  

В отчете отображаются как сами показатели, так и их прирост (или падение) относительно предыдущего периода – Динамика (в процентах). Динамика рассчитывается по следующей формуле:

  • Показатель текущего периода – показатель предыдущего периода / Показатель предыдущего периода.

Если показатель предыдущего периода равен нулю, то прирост – 100 %.
 
Пользователь может просмотреть расширенную статистику по показателю на диаграмме, задав необходимое количество интервалов времени. При этом на панели рассчитается и отобразится общий период отображения данных.
Диаграммы можно вызвать нажатием на соответствующий показателю заголовок столбца таблицы (диаграмма). При этом в новом окне отобразятся две диаграммы: на круговой диаграмме – заданная пользователем структура показателя текущего периода, а на гистограмме с накоплением – изменения показателя в заданной структуре в течение заданного количества интервалов времени.
