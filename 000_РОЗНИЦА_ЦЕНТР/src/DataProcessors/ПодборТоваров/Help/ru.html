Обработка "Подбор товаров"

Обработка используется для подбора товаров в табличную часть документа . В диалоговом окне подбора можно установить различные варианты отбора товаров: по иерархии номенклатуры, по виду номенклатуры, по сегменту номенклатуры. В соответствии с выбранным вариантом отбора будет показан список групп номенклатуры, список видов номенклатуры или список сегментов номенклатуры.

В правой части диалогового окна будет показан список конкретных товаров, принадлежащих указанной в левой части группе номенклатуры, виду или сегменту номенклатуры. В диалоговом окне подбора можно установить дополнительный отбор по произвольным словам поиска.

  • Если необходимо в списке подбора видеть только товары, которые есть в наличии на указанном в документе складе, то установите флаг Только в наличии.
  • Для просмотра остатков на других складах и ожидаемых поступлениях используйте кнопку Остатки.
  • Если необходимо показывать в подборе только те товары, по которым назначены цены, то установите флаг Только с ценами.
  • Для просмотра других цен по товару используйте кнопку Цены.

При подборе товаров, для которых определен список дополнительных свойств (характеристик, дополнительных сведений, дополнительных реквизитов) можно использовать возможность дополнительного отбора по свойствам. Для реализации такого отбора необходимо установить вариант отбора По виду номенклатуры. В появившемся диалоговом окне будет показан список всех зарегистрированных свойств для выбранного вида номенклатуры. Для отбора нужного товара нужно установить флажки для нужных значений свойств и нажать на кнопку Применить.

Если учет номенклатуры ведется в разрезе характеристик, то после выбора такой номенклатуры программа предложит выбрать и конкретную характеристику. В случае ведения учета номенклатуры в разрезе характеристик выбор номенклатуры без характеристики невозможен.

Подбор товаров может быть произведен с запросом количества и цены или без запроса количества и цен (кнопка Запрашивать количество и цену). При подборе товаров можно показывать список подобранных позиций или отключить этот список (кнопка Показывать подобранные позиции). Для переноса подобранных позиций в документ используется кнопка Перенести в документ.

При подборе товаров можно использовать следующие горячие клавиши.

  • Alt+1 – переход к строке поиска.
  • Alt+Ctrl+1 – установить вариант отбора по иерархии номенклатуры.
  • Alt+Ctrl+2 – установить вариант отбора по виду номенклатуры.
  • Alt+Ctrl+3 – установить вариант отбора по сегменту номенклатуры.
  • Alt+2 – перейти к отбору.
  • Alt+3 – перейти к списку товаров.
  • Alt+4 – перейти к списку подобранных товаров.

Переходы также можно осуществлять с помощью клавиши Tab.
